    input

    The trial input attribute lists the input types that the player is familiar with. Each input element must include both a handle and an on propery.
    The handle is the way we refer to this type of input inside the player (in the interaction section), it is basicly the name we give this input. The on parameter describes the specific input that triggers this element.
